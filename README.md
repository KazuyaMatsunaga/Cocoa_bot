# Cocoa_bot
アニメ「ご注文はうさぎですか？」の登場キャラのココアちゃんのbotです

## 主な機能
・「ランチ食べたい」と入力するとその日のランチを決めてくれます．
(ランチのリストは法政大学小金井キャンパス周辺のお店の中で学生がよく行くお店となっています)
